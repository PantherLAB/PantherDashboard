To make this project work as the public release copy the "DQMH Scripting Plugings" folder to 
C:\Users\<USER>\Documents\LabVIEW Data
This is the default location where DQMH Framework will look for DQMH Plugings, any edition to the plugins folder should be made on the Panther Dashboard project, to test the changes, copy again the plugings folder to LabVIEW Data folder to reflect the changes.
